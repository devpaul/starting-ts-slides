Getting Started

What is TypeScript?

  • A typed superset of JavaScript
  • A language transpiler

Basic Types

Primitive Types

  • boolean
  • string
  • number
  • any
  • void

				var yes: boolean = true;

				var name: string = 'I am a string';

				var value: number = 10;

				var placeholder: any = 'I can be anything';

Non-Primitive Types

  • array
  • object
  • tuples
  • enum
  • function
  • class

				var names: string[] = [ 'Paul', 'Dylan' ];

				var values: Array<number> = [ 10, 20, 30 ];

				var tuple: [ number, number, number ] = [ 10, 20, 30 ];

				var obj: object = { src: 'url' };

Functions


				const greeting = function (thing: string): void {
					console.log(`Hello ${ thing }`);
				}

				function average(... values: number[]): number {
					const sum = values.reduce((total, value) => total += value);
					return sum / values.length;
				}

Interfaces

Interfaces represent contracts.

These contracts guarantee the shape an object and what it returns

Generic Objects


				interface Person {
					firstName: string;
					lastName?: string;
					age: number;
				}

				const user: Person = {
					firstName: 'Paul',
					lastName: 'Shannon',
					age: 38
				};

Functional Objects


				interface Convert {
					(value: number): string
					(value: number, digits: number): string;
					(value: string): string;
				}

				const method: Convert = function (value: number | string, digits?: number) {
					if (typeof value === 'string') {
						return value;
					}
					else {
						return value.toFixed(digits);
					}
				}

Generics


				interface Bag<T> {
					data: T;
				}

				const numberBag: Bag<number> = { data: 10 };
				const stringBag: Bag<string> = { data: 'str' };

Ambient Declarations

All values are comprised of two components

A declaration that describes a variable (e.g. name and type)

A definition that concretely explains that symbol (i.e. code)

Ambient Declarations are external, declaration-only files that allow you to safely use existing JavaScript libraries with TypeScript

TypeScript Transpiler

Transpiles modern JavaScript

  • class
  • for..of
  • spread
  • destructuring
  • async..await

ES6 Class


				class Person {
					readonly name;
					private birthdate: Date;

					constructor(name: string, birthdate: Date) {
						this.name = name;
						this.birthdate = birthdate;
					}

					get age(): number {
						return Date.now() - this.birthdate.getTime();
					}
				}

				const p1 = new Person('Paul', new Date('03/20/1979'));

ES5 Constructor Function


				var Person = (function () {
					function Person(name, birthdate) {
						this.name = name;
						this.birthdate = birthdate;
					}
					Object.defineProperty(Person.prototype, "age", {
						get: function () {
							return Date.now() - this.birthdate.getTime();
						},
						enumerable: true,
						configurable: true
					});
					return Person;
				}());
				var p1 = new Person('Paul', new Date('03/20/1979'));

TypeScript's Benefits

Interfaces create clear intent


				function myFunction (): number {
					const value = Math.random();

					// Somewhere in a 1000 line function ...
					if (value > 0.99) {
						// This produces an error!
						return 'gotcha!';
					}
					return value;
				}

Code is easier to maintain

because the TypeScript compiler will warn you of incompatible APIs

IDE integration

makes writing to the interface easier and removes guesswork

Fewer mistakes

means more time writing code and less time debugging
